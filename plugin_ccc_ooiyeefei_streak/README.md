# CCC Skills — Streak

A universal challenge tracker plugin by [ooiyeefei](https://github.com/ooiyeefei/ccc).

Track any personal challenge—learning, fitness, building, creative, or habits—with intelligent cross-challenge insights.

## Source

```
https://github.com/ooiyeefei/ccc/tree/main/skills/streak
```

## Skills

| Skill | Description |
|-------|-------------|
| `streak` | Universal challenge tracker with flexible cadence and cross-challenge learning detection |

## Commands

| Command | What it does |
|---------|--------------|
| `/streak` | Log progress to active challenge |
| `/streak-new` | Create new challenge with guided setup |
| `/streak-list` | Display all active and paused challenges |
| `/streak-switch NAME` | Change active challenge |
| `/streak-stats` | Show progress statistics |
| `/streak-insights` | Discover cross-challenge patterns |
| `/streak-pause NAME` | Pause a challenge |
| `/streak-archive NAME` | Archive a challenge |
| `/streak-resume NAME` | Resume paused/archived challenge |
| `/streak-telegram` | Deploy interactive Telegram bot |
